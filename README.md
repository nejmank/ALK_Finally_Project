# ALK_Finally_Project
## Automatyzacja strony **www.komputronik.pl**
Projekt miał na celu automatyzację pięciu przypadków testowych napisanych dla strony komputronik.pl

Wykorzystane frameworki:
- Selenium WebDriver
- TestNG
- WebDriverManager
- ExtentReports
- JUnit
- JSON

Środowisko na jakim został napisany projekt:
- IntelliJ Aqua
- System operacyjny Linux Ubuntu
